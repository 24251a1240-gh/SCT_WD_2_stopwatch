⏱️ Modern Stopwatch Web Application

A modern, responsive, and interactive Stopwatch Web Application developed using HTML, CSS, and JavaScript as part of "SkillCraft Technology Internship - Task 02."

The application provides accurate time tracking with essential stopwatch functionalities and a clean, user-friendly interface.



 🚀 Features

- ▶️ Start Stopwatch
- ⏸️ Pause & Resume Timer
- 🔄 Reset Stopwatch
- 🏁 Record Unlimited Lap Times
- ⌨️ Keyboard Shortcuts
- 📱 Fully Responsive Design
- 🌙 Modern Glassmorphism UI
- ✨ Smooth Animations & Transitions
- ⏱️ Live Digital Clock
- 📊 Lap Counter & Status Indicator


 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
